2. 基本语法

2.1 HelloWorld

// public的类名必须与文件名大小写一致
public class HelloWorld {
  // 程序的执行入口
  public static void main(String[] args) {
    System.out.println("Hello world!");
  }
}

执行过程：

  1. 编写源码文件（.java），如：HelloWorld.java
  2. 编译器编译生成字节码文件（.class），如：javac HelloWorld.java
  3. jvm执行字节码文件，如：java HelloWorld.java

2.2 数据类型

2.2.1 基本数据类型

4类8种

  • 整型
    • byte：一个字节
    • short：二个字节
    • int：四个字节（默认）
    • long：八个字节
  • 浮点型
    • float：四个字节
    • double：八个字节（默认）
  • 布尔型：
    • 只有两个值 true或false
    • boolean：一个字节
  • 字符型：
    • 如’A'、‘B’
    • char：两个字节

注意事项：

  1. long类型赋值时需要在常量后面加上“L”或“l”；float类型赋值时需要在常量后面加上“F”或“f”；
  2. byte与short赋值时不能超过其值范围
  3. 字节大小与值范围没有直接关联，如float类型能表示的最大值超过long类型
  4. 浮点数都是近似数，精度有限，如用浮点数没精确法表示1/3
// long类型在赋值时需要在当前值后面加上“L”作标识
long numLong = 1L;
// float类型在赋值时需要在当前值后面加上“F”作标识
float numFloat = 1.0F;

// 编译器类型转换优化
short s = 1;
// 如果byte、short、char右侧的值没有超过范围，编辑器会自动进行强制类型转换

short s1 = 1;
short s2 = 2;
// short s3 = s1 + s2; // 编译不通过
// 编译器常量优化
short s4 = 1 + 2; // 编译器会将所有常量的计算提前进行（只有不存在变量）

2.2.2 引用数据类型

  • 字符串
  • 数组
  • 类
  • 接口
  • lamada

数组

// 1. 数据类型[] 数组名称 = new 数据类型[数组长度]
int[] arr = new int[10];

// 2. 数据类型[] 数组名称 = new 数据类型[] { 元素1, 元素2, ... };
int[]arr = new int[]{1, 2, 3, 4};

2.2.3 变量与常量

  • 常量：程序运行过程中不能修改
  • 变量：程序运行过程中可以修改
// 1. 声明： 数据类型 变量名称;
int num;
// 2. 赋值：变量名称 = 具体值;
num = 1;
// 3. 声明 + 赋值：数据类型 变量名称 = 具体值;
int num = 1;

注意事项

  • 未赋值的变量不能使用，否则会报错

类型转换

  • 自动类型转换（隐式）
  • 强制类型转换（显式）
byte numByte1 = 0;
// 自动类型转换（将范围小的数据类型转换为范围大的数据类型）
short numShort = numByte1;
float numFloat = 100L

// 强制类型转换（将范围大的数据类型转换为范围小的数据类型）
// 可能会发生 “数据溢出” 或 “精度损失”
// 非必要情况下不建议使用。
byte numByte2 = (byte) numShort;
int numInt1 = 1000;
byte numByte3 = (byte) numInt1; // -24，数据溢出
int numInt2 = (int) 3.5; // 3，精度损失

// byte、short、char可以发生数学运算，例如：“+”，在运算时先提升为int类型再进行计算
char ch1 = 'A';
System.out.println(ch1 + 1); // 66

// boolean不能发生数据类型转换

2.2.4 ASCII码表

  • 0x30 => 对应 0 (十进制值:48)
  • 0x41 => 对应 A (十进制值:65)
  • 0x61 => 对应 a (十进制值:97)

2.3 基本运算符

  • 算术
    • +、-、* 、/、%
    • ++、–（前自增/减、后自增/减）
  • 赋值
    • =
    • +=、-=、*=、/=、%=（隐含强制类型转换）
  • 比较
    • >、>=、<、<=、==、!=
  • 逻辑（参与运算的、结果都是boolean类型）
    • &&
    • ||
    • ！
    • && 或 || 短路（即如果根据左边表达式的值，已经能够知道最终结果，右边的表达式不会执行）
  • 三元
// 赋值运算符+=、-=、*=、/=、%= 隐含强制类型转换；
short num = 5
num += 5;
// 等价于 num = (short) num + 5;
 
// 三元运算符：需要三个数据才可进行的运算。
// 数据类型 变量名称 = 条件判断 ？ 表达式A : 表达式B;
int num = 1 > 100 ? 1 : 100;
// 变量的数据类型 需要 与表达式A、表达式B的数据类型对应
System.out.println(1.5 > 100 ? 1.5 : 100;)

“+”的三种用法：

  • 数值相加
  • 字符类型运算（转换成int进行相加）
  • 字符串的运算（字符串连接）

算术运行符中，自增/减运行符（++/–）在“单独作使用”（如i++），运行符前后顺序没有影响，而在“混合使用”（如int j = ++i）这两种情况有所区别，“前自增/减”会先修改变量的值再处理其它操作；而“后自增/减”会先处理其它操作再修改变量的值

2.5 流程的控制

  • 顺序结构
  • 选择结构
    • if / if-else / if…else if…else
    • switch
  • 循环结构
    • for / while / do-while

2.5.1 if

if (关系表达式) {
  语句体;
}

int age = 15;
if (age > 30) {
  Sytem.out.println("中年");
} else if (age > 18) {
  Sytem.out.println("青年");
} else {
  Sytem.out.println("少年");
}

2.5.2 switch

switch(表达式) {
  case 常量值1:
    语句体1;
    break;
  case 常量值2:
    语句体2;
    break;
  ...
  default:
    语句体n + 1;
    break;
}
  • 向下执行直到遇到break，或结束
  • switch中case的条件唯一（不能重复），否则编译不通过
  • 表达式的结果只能是：
    • 基本数据类型中的byte、short、int、chart
    • 引用数据类型中的String、emnum枚举
  • case顺序可以随机，break可以省略（向下继续执行）

2.5.3 循环

执行流程：
  1.初始化表达式
  2.条件判断（布尔表达式）
  3.循环体
  4.步进表达式


for (初始化表达式; 条件判断; 步进表达式) {
  循环体;
}


初始化表达式
while (条件判断) {
  循环体;
  步进表达式;
}


初始化表达式
do {
  循环体;
  步进表达式;
} while (条件判断);
  • 次数确定多用for循环，次数不定多用while循环；
  • 在条件判断为false的情况下，do-while还会执行一次；
1) 循环控制语句
  • break：打断循环
  • continue：继续下次循环
2). 死循环
for (let i = 0; i < 100;) {
  System.out.println("Hello World!");
}
3). 循环嵌套
for (let i = 0; i < 24; i++) {
  for (let j = 0; i < 60; j++) {
    System.out.println(i + ":" + j);
  }
}
