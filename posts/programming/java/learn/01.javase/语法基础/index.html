1. 计算机通识

1.1 进制转换

  • 辗转相除法（十进制转换为其它进制）
  • 位权累加法（其它进制转换为十进制）

1.2 存储单位

  • 1bit = 位（表示0或1）
  • 1Byte = 8bit （数据存储的最小单元）
  • 1KB = 1024Byte
  • 1MB = 1024KB
  • 1GB = 1024MB
  • 1TB = 1024GB
  • …

1.3 Java简介

95年发布，已有20来年的历史。目前大部分公司使用的主要版本还是java8。

java编写的程序是跨平台的，但是JVM是非跨平台的（每种平台都有各自版本的JVM）

  • JDK = 编译器等开发工具 + JRE；
  • JRE = 运行类库 + JVM；

1.3.1 dos

  • 盘符切换，如：D: (${磁盘标识}:)
  • 切换目录 cd
  • 清除 cls （全写：clear screen）
  • 退出 exit

1.3.2

  • 配置JAVA_HOME
  • 配置PATH

mac下无需如此，开发配置简单。

2. 语法

2.1 HelloWorld

// public的类名必须与文件名大小写一致
public class HelloWorld {
  // 程序的执行入口
  public static void main(String[] args) {
    System.out.println("Hello world!");
  }
}

2.1.1 注释

// 单行注释

/*
  多行注释
*/

2.1.2 关键字与标识符

  • 关键字：预先定义，用于特定用途的，如public、class等；
  • 标识符：
    • 由大小写字母（a-zA-Z）、数字（0-9）、美元符号（$）、下划线（_）组成；
    • 不能以数字开头；

2.2 数据类型

2.2.1 基本数据类型

  • 整型
    • byte：一个字节
    • short：二个字节
    • int：四个字节（默认）
    • long：八个字节
  • 浮点型
    • float：四个字节
    • double：八个字节（默认）
  • 布尔型：
    • 只有两个值 true或false
    • boolean：一个字节
  • 字符型：
    • 如’A'、‘B’
    • char：字节数不定
